The **Otic** app is a branch off of the Aural iOS music local file management app I coded in a group for a Mobile Device Programming
implementation project as an Appstate CompSci assignment. As the visionary for this project, I wanted to see 
how difficult it would be to make a customizable filing system for organizing music files as well as music playlist structures. 

Currently, Otic is at a funcitonal state of music playback and playlist creation and management, but the UI can be improved
may more, and I plan to add more playback options such as customizable playback speed and audio effects like pitch and reverb.

Current features include:
* MP3 and WAV support
* Playlist Shuffling and Looping
* Sorting options by name, date created, number of items, or last modified date
* Playlists can have images uploaded from the camera roll for thumbnails

Planned features include:
* Seperate view for currently played tracks
* Cloud storage system for playlist structures
* More playback modification options such as speed and pitch
